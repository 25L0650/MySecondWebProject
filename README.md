Desi DeLights : Food Delivery website

A modern, responsive food-delivery themed website using HTML5 and CSS3, showcasing a desi food brand called Desi DeLights.
It includes a hero banner, menu section, food gallery, FAQs, and an order form with interactive UI elements.

Features:

1. Fully responsive layout
desined with flexible grids and media queries to ensure smooth display on all screen sizes.

2. modern hero section
a bold introductiry banner with background image, tagline, CIA button.

3.interactive menu cards
hover-scaling cards displaying top food items with images and descriptions.

4.image gallery
grid-based photo gallery showing dishes in a clean, aesthetic layout displaying the modern times.

5.functional order form
it includes:
name & email inputs.
checkbos-based dish selection.
radio buttons for delivery type.
city dropdown.
special requirments textarea.

6.expandable FAQs
smooth accordin-style FAQ items creadted using pure CSS(checkbox+sliding sectors), designed beautifully with amazing background color selection.

7.clean and themed UI
it uses:
consistent matt red-black themes.
box shadows, rounded corners.
custo google fonts(poppins).
